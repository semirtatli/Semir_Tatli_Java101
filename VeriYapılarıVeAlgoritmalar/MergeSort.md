# Merge Sort

[16,21,11,8,12,22] -> First state

## Sort

```
- [16,21,11]    [8,12,22]
- [16]  [21,11]  [8]  [12,22]
- [16]  [11,21]  [8]  [12,22]
- [11,16,21]    [8,12,22]
- [8,11,12,16,21,22]
```

## Big O

- Best Case:    **O(nlog(n))**
- Average Case: **O(nlog(n))**
- Worst Case:   **O(nlog(n))**
